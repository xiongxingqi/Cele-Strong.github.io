一 . 终端

1. windows terminal

1.1 快捷键

  • alt + shift + +/- 竖直或水平分割指定窗格
  • ctrl + 逗号 打开设置
  • shift + ctrl + w 关闭当前窗格
  • alt + 方向键 更换选中窗格
  • win + tilda(波浪线) 进入quake模式

此处给出官网地址:,Windows 终端概述 | Microsoft Learn

ps:这里只列举了常用快捷键,更多快捷键可以到terminal查看和和修改
